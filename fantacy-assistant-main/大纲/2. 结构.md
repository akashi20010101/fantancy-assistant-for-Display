- 大模型 (LLM): Gemini 1.5 Flash (高速度、长上下文、高免费额度)

- 服务器 (Serverless): Cloudflare Workers (全球边缘计算，极低延迟)

- 代码仓库 GitHub (云端)

- 开发环境 Codespaces (云端浏览器)

- 逻辑框架: Hono (专为边缘计算设计的超轻量 TypeScript 框架)

- API 入口: Hono Routing (替代传统的 API Gateway，直接集成在 Workers 中)

- 开发语言: TypeScript (Node.js 兼容环境) (强类型保证逻辑严谨)

- 部署框架: Wrangler (Cloudflare 原生工具，一行命令完成全家桶部署)

- 结构化数据库: Cloudflare D1 (原生 Serverless SQL 数据库，存好感度与属性)

- 向量数据库: Cloudflare Vectorize (原生向量库，存少女的长期聊天记忆)

- 文本向量化: Workers AI (@cf/baai/bge-small-en-v1.5) (边缘侧直接向量化，速度更快)

- 文件存储: Cloudflare R2 (替代 S3，核心优势：0 流量费，存模型与语音)

- 向量化框架: LangChain.js (负责编排 RAG 流程与记忆检索)

- 跨平台框架: Flutter (Dart) (负责 App 端 UI 与 Live2D 交互展示)

- 身份验证: Clerk (极简接入的第三方登录方案，支持多端同步)
